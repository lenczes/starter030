When to Pull From GitHub?

When should you pull code from your repository, and why?

When you have been working on a local copy of your project on your computer, and someone modifies the remote repository before you execute your next run, commit, push Git commands.

The conflict happens when you try to run the git push -u origin master command. This will check the files in the GitHub repo and look at any files that have been changed.
